# Light Curves Collection from the Personal Archives
This repository contains resource descriptors for publishing light-curve collections.
The collection originates from the personal archives.

We publish this archive with the permission of the owners and their colleagues, 
who are convinced that archival observations should be accessible to the community.

The time series were derived from CCD images obtained with various small telescopes 
at different locations.
Most observations were carried out to monitor selected objects.

The time series were produced using a custom photometric pipeline.
Differential photometry was performed using comparison star magnitudes based on Landolt standards.

Each photometry point also contains a field that encodes the site and telescope at which observations were carried out.
